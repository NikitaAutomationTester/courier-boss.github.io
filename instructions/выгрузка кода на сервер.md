#  Выкладка кода на сервер без обновления базы данных

На macOS
2. Удаляем старый архив на компьютере



На macOS (терминал)
2. Собираем архив проекта на Mac

cd /Users/nikitasmatko/Documents && zip -r BBEx.zip BBEx -x "*/node_modules/*" -x "*/.git/*" -x "*/server/.env" -x "*/server/data/*"

* Указываем файлы которые игнорируются чтобы на сервере не стёрлась база данных

На macOS (терминал)
3. Заливаем архив на VPS (сервер)

scp /Users/nikitasmatko/Documents/BBEx.zip root@88.218.63.85:/root/


На VPS (терминал)
4. Заходим на сервер

ssh root@88.218.63.85
Вводим пароль


На VPS (терминал)
5. Распаковываем архив на сервере 

cd /root
unzip -o BBEx.zip

На VPS (терминал)
6. Удаляем после архив с сервера (чтобы на захламлять)

rm /root/BBEx.zip


На VPS (терминал)
7. Обновляем зависимости

cd /root/BBEx/server
npm install

Для чего: подтянуть пакеты после изменений в `package.json` и подхватить новый код.


На VPS (терминал)
8. Перезапускаем приложение

pm2 restart bbex
